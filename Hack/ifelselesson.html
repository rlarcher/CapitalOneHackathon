Lesson: For Loops

The newest X-Men movie is coming out on Friday and you?re planning to see it with your friends: Steve, Tameka, Chen, and Sonali. You know that Tameka, Chen, and Sonali can definitely make it on Friday. Unfortunately, Steve got a bad grade on the last math test and might be grounded. You?re responsible for buying the movie tickets. How many tickets will you buy? Well, it depends. If Steve is grounded, then he can?t come, so only 4 of you need tickets. But if Steve can come, then you will need 5. Steve?s freedom will determine how many tickets you get.

You can model your decision-making like this:
if (Steve is grounded)
{
buy 4 tickets;
}
else
{
buy 5 tickets;
}

So what does this code say? Let?s look at it bit by bit. First:
if (Steve is grounded)
{
buy 4 tickets;
}

You can think of this phrase as basically saying, ?If it is true that Steve is grounded, then do what?s inside these braces.? In this case, the stuff inside the braces is to buy 4 tickets. But what happens when isn?t true that Steve is grounded? Then you completely ignore what is inside the first set of braces and jump to
else
{
buy 5 tickets;
}

Now let us consider the same situation but suddenly your another friend Tim wants to join too. However, his parents might take him to a concert instead. Therefore, now you have to add details about Tim in your if/else statement too. You?re once again responsible for buying the movie tickets. Since Tim wants to join you, how many tickets will you need to buy? Well, it depends. If Steve is grounded and Tim?s parents decide to take him to a concert instead, then both can?t come, so only 4 of you need tickets. But if Steve can come or if Tim can come, then you will need 5. If both Steve and Tim can come, then you will need 6 tickets. Steve?s freedom and Tim?s parent?s decision will determine how many tickets you get.

You can model your decision-making like this:
if (Steve is grounded and Tim?s parents decide to take him to a concert)
{
buy 4 tickets;
}
else if (Steve is grounded or Tim?s parents decide to take him to a concert)
{
buy 5 tickets;
}
else()
{
buy 6 tickets;
}

So what does this code say? Let?s look at it bit by bit. First:
if (Steve is grounded and Tim?s parents decide to take him to a concert)
{
buy 4 tickets;
}

You can think of this phrase as basically saying, ?If it is true that Steve is grounded and it is true that Tim?s parents decide to take him to a concert, then do what?s inside these braces.? In this case, the stuff inside the braces is to buy 4 tickets.

But what happens when either Steve of Tim can come to watch the movie? Then you completely ignore what is inside the first set of braces and jump to
else if (Steve is grounded or Tim?s parents decide to take him to a concert)
{
buy 5 tickets;
}

Note that instead of using else(), we use else if() this time for our second statement. The reason for this is because now we have to look at another condition, which is whether either Steve or Tim can come to the movie. If Steve is not grounded and Tim?s parents don?t decide to take him to a concert, then both can come. In this situation, you can ignore both the first and second set of braces and jump to the third set
else()
{
buy 6 tickets;
